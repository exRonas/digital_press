# Руководство оператора

## Система "Цифровой банк печати"

Данное руководство предназначено для операторов системы, ответственных за загрузку и управление газетными архивами.

---

## 1. Вход в систему

1. Перейдите по адресу: `https://ваш-сайт/admin/login`
2. Введите **логин** и **пароль**, полученные от администратора
3. Нажмите **"Войти"**

После входа вы попадёте в панель управления.

---

## 2. Загрузка выпуска газеты

### Шаг 1: Подготовка файла
- Формат: **PDF**
- Максимальный размер: **500 МБ**
- Рекомендуемое качество сканирования: **300 DPI** (для лучшего OCR)

### Шаг 2: Загрузка
1. В боковом меню выберите **"Загрузить выпуск"**
2. Выберите **издание** из списка (например, "Казахстанская правда")
3. Укажите **дату выпуска**
4. Введите **номер выпуска**
5. Выберите **язык**:
   - Русский
   - Казахский
   - Смешанный
6. Перетащите PDF-файл в область загрузки или нажмите для выбора
7. Нажмите **"Загрузить"**

### Шаг 3: Обработка
После загрузки система автоматически:
- Сохранит файл в защищённое хранилище
- Запустит **OCR-распознавание** текста
- Проиндексирует содержимое для поиска

Документ сразу появится в публичном каталоге.

---

## 3. Управление документами

### Просмотр списка
- Раздел **"Документы"** в боковом меню
- Используйте **поиск** для фильтрации по названию
- Используйте **фильтр по языку**

### Действия с документом
Нажмите на меню (⋮) справа от документа:

| Действие | Описание |
|----------|----------|
| **Просмотр** | Открыть PDF в новой вкладке |
| **Редактировать** | Изменить номер, дату, язык |
| **Повторить OCR** | Перезапустить распознавание текста |
| **Удалить** | Удалить документ (необратимо!) |

### Редактирование
Можно изменить:
- Номер выпуска
- Дату выпуска  
- Язык документа

**Нельзя изменить:** издание, PDF-файл (нужно удалить и загрузить заново)

## 5. Полнотекстовый поиск

### Как это работает
1. При загрузке PDF система распознаёт текст (OCR)
2. Текст индексируется для быстрого поиска
3. Пользователи могут искать по содержимому газет

### Если поиск не работает
- Проверьте качество скана (нечёткий текст распознаётся хуже)
- Используйте **"Повторить OCR"** для документа
- Обратитесь к администратору

---

## 6. Частые вопросы

### Не загружается файл
- Проверьте формат (только PDF)
- Проверьте размер (до 500 МБ)
- Попробуйте другой браузер

### Документ загрузился, но не появился в каталоге
- Обновите страницу
- Подождите 1-2 минуты (идёт обработка)

### Как изменить издание у документа?
Нельзя изменить издание. Нужно удалить документ и загрузить заново с правильным изданием.

### Как узнать статус OCR?
В разделе "Документы" рядом с документом может быть индикатор статуса обработки.
